# How to start

Make sure you have [Jekyll installed](https://jekyllrb.com/docs/), then run:

```bash
# Install dependencies
bundle install

# Start local server
bundle exec jekyll serve --livereload
```

# How to deploy

Deployment is taken care of by GitHub Pages.
